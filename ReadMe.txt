The Randomizer is a self contained plugin that allows a WordPress administrator to display a list of random posts in the footer of every page or post. This presents the visitor with a new selection posts each time a page is loaded.

This Software was written by T-roy http://trialsoftwarez.com. Related information and original idea was inspired by Lance http://lancelhoff.com

The Randomizer is considered Open Source. Please see the included GPU GPL License for more information.


How To install and use The Randomizer:

1. Extract the contents of the Randomizer.zip
 
2. Copy the Randomizer folder to your WordPress /wp-content/Plugins directory

3. Enable The Randomizer from your Plugins menu in the WordPress administration panel

4. Enjoy the randomly generated list of posts displayed in the footer
